ПОЛЬЗОВАТЕЛЬСКОЕ СОГЛАШЕНИЕ
Security Shield X - Система защиты информации

1. ОБЩИЕ ПОЛОЖЕНИЯ
1.1. Security Shield X (далее - "Программа") является экспериментальной 
    системой защиты информации, разработанной в образовательных целях 
    для участия в олимпиаде по информационной безопасности.
1.2. Программа не является коммерческим продуктом и не предназначена 
    для использования в производственных средах.

2. ЦЕЛЬ РАЗРАБОТКИ
2.1. Программа демонстрирует методы обнаружения самописного вредоносного 
    программного обеспечения с использованием анализа поведенческих сигнатур.
2.2. Основная цель - показать возможность обнаружения неизвестных угроз 
    до их внесения в сигнатурные базы антивирусных систем.

3. ОГРАНИЧЕНИЯ ОТВЕТСТВЕННОСТИ
3.1. Разработчик не несет ответственности за:
    - Ложные срабатывания системы
    - Недетектирование реальных угроз
    - Любые повреждения системы пользователя
    - Потерю данных
3.2. Программа предоставляется "КАК ЕСТЬ" без гарантий любого рода.

4. СБОР ДАННЫХ
4.1. Программа анализирует только метаданные процессов и файлов
4.2. Не передает пользовательские данные на внешние сервера
4.3. Все аналитические операции выполняются локально

5. УСЛОВИЯ ИСПОЛЬЗОВАНИЯ
5.1. Разрешено использование только в тестовых средах
5.2. Запрещено использование для защиты критически важных систем
5.3. При обнаружении угроз рекомендуется проверка другими средствами

6. ИССЛЕДОВАТЕЛЬСКАЯ СОСТАВЛЯЮЩАЯ
6.1. Программа использует следующие методы анализа:
    - Анализ энтропии файлов
    - Поведенческие сигнатуры
    - Статистические аномалии
    - Эвристический анализ
6.2. Все алгоритмы открыты для изучения и модификации
